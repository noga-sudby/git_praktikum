pwd выводит путь к текущей директории.
cd перейти к домашней директории
ls Вывести содержимое директории
cd имя_папки Сменить директорию (если в названии папки есть пробелы, при вводе нужно использовать кавычки)
.. Чтобы вернуться в родительскую директорию — то есть на уровень выше
ls с флагом -a  отобразит все и скрытые
ls ~ выведет содержимое домашней директории вне зависимости от того, что показывает pwd
ls .. покажет содержимое родительской директории

touch %ИМЯ_ФАЙЛА% создать файл
$ touch my-new-file.txt # создали файл my-new-file.txt 

mkdir создания директорий 
$ mkdir new-dir # создали директорию new-dir 
=====
$ mkdir -p dir1/dir-inside/dir-deeper-inside
# создали папку dir-deeper-inside в папке dir-inside, которая находится в папке dir1 

mkdir ~/my-git-projects создаст папку my-git-projects внутри домашней директории

touch ../../file.txt создаст файл file.txt на две папки выше по иерархии

cp что_копируем куда_копируем
=========
$ cp index.html src/
# скопировали index.html в папку src 
========
$ cp что_копируем что_копируем что_копируем куда_копируем
=======
$ cp -r для копирования папок

mv перемещение
========
$ mv table.csv ./very-important-files
# сначала указываем имя файла, который хотим переместить, потом путь — куда перемещаем 

cat Чтобы прочитать файл
работает только с текстовыми файлами

rm удалить файл
$ rm example.txt # удалили файл example.txt из текущей папки 
========
rmdir Удалить папку 
$ rmdir images # команда удалит папку images из текущей директории, 
               # если папка images пуста 
=======
Если папку всё-таки нужно удалить вместе со всем её содержимым
$ rm -r images # удалили папку images со всем её содержимым из текущей директории 

сразу несколько команд
$ mkdir second-project && cd second-project && touch index.html style.css
# создаём папку second-project,
# переходим в папку second-project
# и создаём в ней два файла: index.html и style.css 

автозаполнение
вспомнить, с каких букв она начинается. Можно набрать их в командной строке и дважды нажать клавишу Tab
==========
Если ввести cd с названием папки, а затем нажать Tab, в консоль в качестве подсказки выведутся все возможные пути.

cd c:/ или cd /c Чтобы попасть в корневую директорию Windows
